	2. В разработке игра между двумя игроками, с отправкой данных о действиях на сервер
	#### **Руководство по сборке и запуске проекта**
	1. Клонирование репозитория
		```
		$ git clone https://github.com/MacloMen336/rus_blackjack
		```
 
	1. Запуск программы
		```
		$ python __main__.py
		```
